Hardware Validation & PCB Design

Bed of Nails V3
Test Fixture

I spearheaded the full hardware lifecycle of a 2-layer test fixture, progressing from LTSpice simulation and breadboard prototyping to Altium layout and Python validation.

During my time at Electrans Technology, I engineered a custom window comparator circuit that automated 100% of termination checks. This project involved deep hardware design, ensuring that the physical test fixture provided reliable, repeatable results for production units.

The workflow included validating the circuit via Python scripts and managing the PCB layout in Altium Designer to ensure mechanical alignment with pogo-pins for efficient board testing. This automation significantly reduced manual testing time and increased the throughput of the manufacturing line.

Key Technologies

  • Altium Designer: PCB Layout and Component Selection
  • LTSpice: Circuit Simulation and Window Comparator Design
  • Python: Automated Data Collection and Validation Scripts
  • Hardware Testing: Pogo-pin Mechanical Alignment and Signal Integrity Checks
