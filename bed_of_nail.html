Hardware Validation & PCB Design

Bed of Nails V3
Test Fixture

Spearheaded the full hardware lifecycle of a 2-layer test fixture at Electrans Technology — from LTSpice simulation and breadboard prototyping through to Altium layout and Python automation. Engineered a custom window comparator that achieved 100% termination check coverage.

PCB Design Altium Designer Python LTSpice Hardware Testing
100% termination checks automated 2-layer PCB design Zero defects in shipped hardware

During my co-op at Electrans Technology, I was tasked with designing a production-grade test fixture from the ground up. The Bed of Nails V3 is a 2-layer PCB that uses spring-loaded pogo pins to make simultaneous contact with a device under test (DUT), enabling fast, repeatable production-line validation.

The project began with LTSpice simulation to validate the window comparator circuit — a critical design that checks whether each termination resistor on the DUT falls within an acceptable tolerance window. This gave me confidence in the circuit before committing to physical hardware.

From there, I prototyped the comparator on breadboard to verify real-world performance matched simulation. Once validated, I performed the full PCB layout in Altium Designer, carefully managing mechanical constraints to align pogo pins with the DUT footprint and ensuring signal integrity on the comparator outputs.

The final piece was a Python validation script that communicates with the fixture, reads comparator outputs, and programmatically validates 100% of termination checks — eliminating manual probing entirely. The result: a reproducible, zero-defect test process that shipped directly to the production line.

Key Technologies

  • Altium Designer: Full 2-layer PCB layout, component footprint creation, mechanical DRC for pogo-pin alignment
  • LTSpice: Window comparator design and simulation — verified tolerances across temperature and component variation
  • Python: Automated serial/GPIO communication with test fixture; pass/fail logging for production line integration
  • Breadboard Prototyping: Iterative hardware validation of comparator circuit before PCB commitment
  • Hardware Testing: Signal integrity checks, pogo-pin alignment validation, DUT interface verification
