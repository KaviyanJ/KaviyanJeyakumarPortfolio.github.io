OptiBOM
System Design

I architected a hybrid procurement automation tool using Node.js and Google Apps Script to solve complex supply chain challenges. By integrating Digi-Key and Mouser APIs, I reduced manual BOM analysis time by over 90%.

OptiBOM was designed to automate the tedious process of validating electronic component stock and lifecycle status. The system performs real-time stock validation and End-of-Life (EOL) detection, ensuring that the engineering team can identify high-risk components before production begins. This project showcases my ability to bridge software automation with practical hardware production needs.
