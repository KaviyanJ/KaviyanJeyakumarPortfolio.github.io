RescueTouch Emergency
System

I designed and developed a two-board emergency system using STM32 microcontrollers, handling both the schematic design in KiCad and the C++ firmware development. The project's main goal was to create a robust assistive device by establishing reliable UART communication between a Braille-labeled input board and an LCD output board.
