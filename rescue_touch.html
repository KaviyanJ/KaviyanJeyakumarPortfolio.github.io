RescueTouch
Emergency System

Designed a two-board STM32-based emergency communication system with KiCad schematics and C++ firmware. Established reliable UART communication between a Braille-input board and an LCD alert board for accessible, time-critical emergency signalling.

STM32 Embedded C++ UART KiCad
Dual STM32 architecture Reliable UART inter-board comms Full KiCad schematics

RescueTouch is a two-board embedded system designed for accessible emergency alerting. The system consists of a Braille-labeled input board that monitors for a distress trigger, and an LCD alert board that receives the signal and displays the emergency alert — the two boards communicating over UART.

The hardware design was done entirely in KiCad, covering both schematics and PCB layout for both boards. Key design considerations included reliable power distribution, proper UART level-shifting between boards, and ensuring the microcontroller reset and boot configurations were correctly handled for reliable startup.

The C++ firmware for both STM32 boards was written using the HAL library. The input board monitors GPIO inputs with software debouncing, and transmits a structured alert packet over UART. The alert board receives the packet, validates it via checksum, and drives the LCD output. The use of a simple packet protocol (start byte, command byte, checksum) ensures reliable communication even in the presence of UART line noise.

This project reinforced the discipline of embedded systems design — where hardware and firmware constraints are tightly coupled, and reliability is non-negotiable for a safety-critical application.

Key Technologies

  • STM32 (ARM Cortex-M): Microcontroller programming using STM32CubeIDE and HAL library for both boards
  • Embedded C++: UART packet protocol design, GPIO interrupt handling, software debouncing, LCD driver
  • UART Communication: Inter-board serial protocol with structured packet framing and checksum validation
  • KiCad: Full schematic capture and PCB layout for both the Braille input and LCD output boards
